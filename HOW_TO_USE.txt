ğŸ”§ TROUBLESHOOTING
================================================================

Problem: "Python is not installed"
Solution: Install Python from https://www.python.org/
          Make sure to check "Add Python to PATH" during installation

Problem: "Dependencies not installed"
Solution: Run setup.bat again

Problem: Login fails
Solution: Check your .env file has correct password
          Make sure Gmail API credentials are in data\ folder

Problem: Browser doesn't open
Solution: Run setup.bat to install Playwright browsers

Problem: Session expires
Solution: Just run run.bat again - it will re-login automatically
